# Cheat Sheet Numbers and Booleans

## Numbers

```js
4
5
4.3
4e2
```

## Numeric Operators

```js
4 + 3   // 7
4 - 3   // 1
4 * 3   // 12
4 / 3   // 1.333
4 % 3   // 1
4 ** 3  // 64
```

## Comparison Operators

```js
4 === 4     // true
'4' === 4   // false
'4' == 4    // true
4 !== 4     // false
4 !== '4'   // true
4 > 3       // true
4 >= 3      // true
4 < 3       // false
4 <= 4      // true
```

## Boolean Operators

```js
a && b // if both a and b are true ==> true, otherwise ==> false
a || b // if both a and b are false ==> false, otherwise ==> true
!a     // if a is true ==> false, if a is false ==> true
```

## if

```js
if (_booleanValue_) {
  // statements
}

if (_booleanValue_) {
  // statements
} else {
  // statements
}
```
