# Cheat Sheet for Types and Loops

## typeof

```js
typeof 4                 // 'number'
typeof 'Robin Hood'      // 'string'
typeof function () {}    // 'function'
typeof true              // 'boolean'
typeof {}                // 'object'
typeof []                // 'object (WAT!)'
Array.isArray([])        // 'true'
typeof undefined         // 'undefined'
typeof null              // 'object'
```

## while && break

```js
while (_boolean_expression_) {
  _expression_body_
}
```

## break

```js
while (_boolean_expression_) {
  //...
  break
  //...
}
```
